# ResolveIT – Smart Grievance & Feedback Management System

ResolveIT is a full-stack complaint management platform where users can submit complaints and track resolution status.

## Tech Stack
- React
- Spring Boot
- MySQL
- REST APIs

## Features
- Complaint submission with attachments
- Anonymous complaint option
- Complaint status tracking
- Admin dashboard
- Staff assignment
- Complaint escalation
- CSV complaint reports

## Roles
Customer – Submit and track complaints  
Staff – Resolve assigned complaints  
Admin – Manage complaints and staff
